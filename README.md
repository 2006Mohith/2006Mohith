## 🧭 About Me

<img align="right" src="https://user-images.githubusercontent.com/74038190/212284158-e840e285-664b-44d7-b79b-e264b5e54825.gif" width="300"/>

I'm an undergraduate in Electronics and Communication Engineering, working toward becoming an **Embedded Systems Engineer** with a focus on embedded software, FPGA, IoT, and AI-powered embedded systems. I like building things that sit close to hardware — where a line of code has to actually talk to a sensor, a board, or a real-time constraint.

- 🔭 **Currently building:** IoT and full-stack projects that combine embedded hardware with real-time web systems
- 🌱 **Currently learning:** STM32, FreeRTOS, CAN Protocol, SPI, UART, I2C
- 🎯 **Goal:** Grow into embedded software and FPGA/VLSI development, with AI-powered embedded systems and robotics as long-term focus areas
- ⚡ **Interests:** Embedded Systems · IoT · Artificial Intelligence · FPGA · Verilog · Computer Architecture · PCB Design · Robotics · Linux · Open Source
- 🌍 **Open Source:** Learning to read, contribute to, and eventually maintain open-source embedded and web tooling
- 🎲 **Fun fact:** I'm equally at home debugging a sensor wiring issue and a race condition in a WebSocket server
